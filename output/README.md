# Team Token Output Directory

This directory contains generated team tokens and data files.

## File Types

- **teams_YYYYMMDD_HHMMSS.json** - JSON format for database seeding
- **teams_YYYYMMDD_HHMMSS.csv** - CSV format for easy distribution to participants

## Usage

1. Generate teams using the script
2. Use JSON file to seed the database
3. Distribute CSV file or individual tokens to teams

Files are automatically timestamped to avoid overwrites.
